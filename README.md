SPA-Angular-Hello-World
=======================

 - To install this repo:

 ```
 git clone https://github.com/rafanog/SPA-Angular-hello-world
 ```
 Install the dependencies:
 
 ```
 npm install
 ```

 - To run:

 Install the server via npm:

 ```
 npm install -g serve
 ```
 
 CD to the repo file
 
 ```
 serve
 ```
